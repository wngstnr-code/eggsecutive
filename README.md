# Eggsecutive

Eggsecutive is a skill-based onchain arcade game where every move matters.  
The goal is simple: cross lanes, stack multipliers, and cash out before the run crashes.

## About The Project

Eggsecutive is built to deliver gameplay that is:

- fast and fun
- competitive but fair
- accessible for new users entering Web3 gaming

It combines casual arcade mechanics with onchain settlement, making it suitable for both everyday players and community-driven competitive events.

## Why Eggsecutive

- Focuses on player decision-making, timing, and risk control
- Lightweight gameplay loop that is easy to learn
- Built on a modular foundation that can grow into leaderboard, tournament, and social features
- Real wallet + onchain flow for transparent outcomes

## Live App

Try it here: https://eggsecutive.vercel.app

## Project Structure

- `frontend/` - Next.js client app (UI, game canvas, wallet flow)
- `backend/` - API and realtime session/game orchestration
- `sc/` - Smart contracts for settlement and related onchain logic
